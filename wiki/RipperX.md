# RipperX

RipperX permet de convertir les CDs en fichiers MP3.

## Configuration

- CDDB: freedb.freedb.org/~cddb/cddb.cgi port 80
- filename format : %# %s
- directory format : %a/%v
- Mp3 :
  - lame MP3 Encoder (installer 'lame')
    - Use VBR (4), 128K
